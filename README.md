# MeliTech

Sistema desenvolvido para monitoramento de colmeias de abelha Jataí, com foco no acompanhamento da temperatura, umidade e vibração da colmeia.

## Sobre o Projeto

O MeliTech foi criado como projeto escolar com o objetivo de unir tecnologia e sustentabilidade no cuidado com abelhas nativas.

A proposta do sistema é acompanhar as condições da colmeia sem a necessidade de abri-la constantemente, reduzindo interferências e facilitando a análise das informações coletadas.

## Objetivo

Monitorar dados importantes da colmeia e auxiliar no acompanhamento da saúde e comportamento das abelhas.

## Dados Monitorados

- Temperatura interna
- Temperatura externa
- Umidade interna
- Umidade externa
- Vibração da colmeia

## Tecnologias Utilizadas

- Arduino
- Sensores de temperatura e umidade
- Sensor de vibração
- HTML
- CSS
- JavaScript
- GitHub

## Espécie Monitorada

Abelha Jataí

Espécie nativa sem ferrão, importante para a polinização e preservação ambiental.

## Funcionalidades

- Leitura em tempo real dos sensores
- Monitoramento interno e externo
- Coleta de dados ambientais
- Acompanhamento da vibração da colmeia
- Visualização das informações em interface web
